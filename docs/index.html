Dependencies

  • Rust & Cargo: Install via Rustup
  • SurrealDB: Follow installation docs
  • Cargo-binstall: See repo instructions
  • Cross: See repo instructions
  • Cast CLI: Monorepo management tool

Essential Commands

# Install Cast CLI (monorepo tool)
cargo install --path ./projects/cast_cli

# Install Cast VSCode Extension
code --install-extension ./projects/cast_vscode/cast.vsix

# Start a work session
cast session start

# Create a new project
cast project new

üîß Operations

Issue Tracking

All issues are tracked as TODO comments in the codebase. Use the TODO Tree VSCode extension to find work items.

Git LFS

Large artifacts (PDFs, binaries) are managed with Git LFS. Ensure you have git-lfs installed.

CI/CD

Automated workflows run on GitHub Actions. Check the Actions tab for build status and deployments.
