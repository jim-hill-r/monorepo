# Blue Eel

A reading education application built with Dioxus that provides comprehensive reading instruction and support for learners of all ages.

## About

Blue Eel is inspired by the mission of blue.eel.education to help everyone develop strong reading skills. The application offers:

- Interactive reading lessons
- Phonological awareness training
- Reading comprehension strategies
- Fluency building exercises
- Vocabulary development activities

## Features

- **Phonological Awareness**: Build foundational skills in recognizing and manipulating sounds in words
- **Reading Comprehension**: Develop strategies for understanding and analyzing written text
- **Fluency Building**: Practice reading smoothly and confidently with appropriate expression
- **Vocabulary Development**: Expand word knowledge and understanding through engaging activities

## Technology

Built with:
- [Dioxus](https://dioxuslabs.com/) - A React-like framework for Rust
- Rust Edition 2024

## Building

To build the application:

```bash
cargo build
```

To run the application:

```bash
cargo run
```

For web deployment:

```bash
dx serve
```

## Target Audience

This application is designed for:
- Students working to improve reading skills
- Individuals with dyslexia or other reading challenges
- Anyone looking to enhance their reading abilities
- Parents and educators supporting readers
